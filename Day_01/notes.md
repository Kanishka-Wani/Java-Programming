### Java is Case Sensative

### How does Java program run ?
 - save with same class name and file name
 - when we run javac filename.java compiler converts the java code into bytecode saved in filename.class file
 - this .class file is given to JVM when you run java filename
 - then JVM generates machine code that will executed by CPU.
 - finally we get intended output
