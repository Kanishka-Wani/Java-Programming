### Variables in Java:
 variable is a named memory location which is used to store and retrieve data.

---

### Datatypes in java:

byte      → 1 byte

short     → 2 bytes

int       → 4 bytes

long      → 8 bytes

float     → 4 bytes

double    → 8 bytes

char      → 2 bytes

boolean   → JVM implementation-dependent

---

Example: int age = 20;
 int --> datatype
 age --> variable name
 value --> 20
